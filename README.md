# שחר פתרונות דיגיטליים - דף נחיתה

דף נחיתה מקצועי ומודרני לעסק "שחר פתרונות דיגיטליים" - המתמחה בפיתוח מערכות ניהול, טפסים חכמים ואוטומציות.

## תכונות ראשיות

- ✨ עיצוב מודרני ומרשים עם אנימציות עדינות
- 📱 רספונסיבי מלא (Desktop, Tablet, Mobile)
- ♿ תומך ב-Accessibility (ARIA labels, semantic HTML)
- 🎨 גרדיאנטים מתקדמים ו-glassmorphism
- 🚀 ביצועים מהירים וקוד אופטימלי
- 📊 טופס יצירת קשר עם חיבור ל-Google Sheets
- 🎯 SEO מותאם (Open Graph, Twitter Cards)

## התקנה ושימוש

### 1. פתיחת הדף

פשוט פתח את הקובץ `index.html` בדפדפן:

```bash
open index.html
```

או גרור אותו לחלון הדפדפן.

### 2. חיבור לGoogle Sheets

לפני ששולחים את הדף לאוויר, יש לחבר את הטופס ל-Google Sheets:

1. פתח את הקובץ `Google_Apps_Script_Setup.md`
2. עקוב אחר ההוראות המפורטות שם
3. העתק את ה-URL שמתקבל לתוך `script.js` בשורה 50

### 3. העלאה לאינטרנט

ניתן להעלות את הדף לכל שירות אירוח סטטי:

- **GitHub Pages** (חינם)
- **Netlify** (חינם)
- **Vercel** (חינם)
- **Cloudflare Pages** (חינם)

או פשוט להעלות ל-cPanel/FTP של האירוח שלך.
